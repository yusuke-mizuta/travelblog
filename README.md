# 旅ブログ

## サイト概要
ブログ投稿サイト

### サイトテーマ
旅で撮った写真をシェアする

### テーマを選んだ理由
DMMwebcampで学んだ機能を使って自分の中で作りたいものとして一番に思いついたのが旅の写真の投稿サイトでした。旅のをしてそこで撮った写真と共にそこで起こった出来事も合わせてシェアできるものがあれば良いなと思い製作しました。

### ターゲットユーザ
旅好きの方

### 主な利用シーン
旅をしたあとの投稿、旅の前の旅行先の参考
